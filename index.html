NCCU Mathematical Sciences Degree Audit System

Transcript import, rule-based graduation audit, administrative course adjustment, and audit result persistence for NCCU Mathematical Sciences undergraduate students admitted in academic years 111–114.

Overview

This repository is the final project for the 114-2 Database Systems course. The system imports transcript JSON files downloaded from iNCCU and evaluates whether a student satisfies the graduation requirements of the NCCU Department of Mathematical Sciences undergraduate program.

Modules

Student Portal

Import transcript JSON files, view imported courses, run degree audits, and review audit results and history.

Admin Portal

Review unresolved courses, create manual course adjustments, query course data, query graduation rules, and view student audit records.

Backend

Express API, Sequelize models, MySQL persistence, transcript import, audit rule evaluation, and result storage.

Frontend

React, Vite, TypeScript, and Tailwind CSS interface for transcript import, audit execution, and result visualization.

專案概要

本專案為 114-2 資料庫系統 期末專案。系統可匯入學生從 iNCCU 下載的 transcript JSON 檔案，並依照政大應數系學士班畢業規則，自動判斷學生是否符合畢業資格。

系統模組

學生端

匯入 transcript JSON、查看已匯入課程、執行畢業審核、查看審核結果與歷史紀錄。

管理員端

查看待確認課程、建立人工調整、查詢課程資料、查詢畢業規則、查看學生審核紀錄。

後端

Express API、Sequelize models、MySQL 資料持久化、transcript 匯入、畢業規則計算與審核結果儲存。

前端

React、Vite、TypeScript、Tailwind CSS，負責 transcript 匯入、畢業審核操作與結果視覺化。
